# 🖥️ System Control & Security Lab

Welcome! In this interactive lab you will work through **7 hands-on steps** that cover core Linux sysadmin and security skills.

## What You Will Learn

| Step | Topic |
|------|-------|
| 1 | Run a Python web server |
| 2 | Inspect its PID and resource usage |
| 3 | Kill and restart the process |
| 4 | Create a dedicated user and application directory |
| 5 | Intentionally misconfigure file permissions |
| 6 | Prove the access risk with a second user |
| 7 | Fix it using the principle of least privilege |

## How This Lab Works

- Each step shows you the **commands to run** with a full explanation of what they do
- Commands with a **▶ Run** button can be executed directly — just click it
- A ✅ **check** runs automatically when you click **Continue** to confirm you completed the step correctly
- If a check fails, re-read the instructions and try again — the terminal is always live

## Prerequisites

- No installation required — you have a full Ubuntu terminal right here
- `python3` and `sudo` are already available
- You are logged in as `root`
